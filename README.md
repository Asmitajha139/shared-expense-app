# Shared Expense App

## Tech Stack
- FastAPI
- Python
- JWT Authentication
- bcrypt
- Pydantic

## Features
- User Registration
- User Login
- Group Management
- Expense Management
- Settlements

## Run Project

pip install -r requirements.txt

python -m uvicorn app.main:app --reload

Open:

http://127.0.0.1:8000/docs
